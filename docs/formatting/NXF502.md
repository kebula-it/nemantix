# NXF502 — Redundant Completion Qualifier

**Category:** Annotation style  

## Rule

When a `@completion:` annotation specifies a state transition where both states are
identical (`X->X`), prefer the single-value form (`X`).

## Rationale

`@completion: frozen->frozen` is semantically equivalent to `@completion: frozen` but
adds noise. The single-value form is the canonical representation.

## Examples

**Compliant:**

```nxs
@completion: frozen
action classify >> classifies intent <<:
  body:
    >> determine category <<
  __body
__action
```

**Non-compliant:**

```nxs
@completion: frozen->frozen
action classify >> classifies intent <<:
  body:
    >> determine category <<
  __body
__action
```
