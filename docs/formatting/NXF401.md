# NXF401 — Import Form

**Category:** Import style  

## Rule

Prefer the **inline** import form when the entire statement fits within 120 characters.
Use the **block** form when the statement would exceed 120 characters.

## Forms

**Inline:**

```nxs
from toolset WebSearch use search, fetch
from toolset FileSystem as fs use read, write, delete
```

**Block:**

```nxs
from toolset WebSearch use:
  search, fetch, index, summarise, rank, filter
__use
```

## Examples

**Compliant — inline (fits within 120 chars):**

```nxs
from toolset WebSearch use search, fetch
```

**Compliant — block (too many tools to fit inline):**

```nxs
from toolset WebSearch use:
  search, fetch, index, summarise, rank, filter, extract, crawl
__use
```

**Non-compliant — block form used when inline would fit:**

```nxs
from toolset WebSearch use:
  search
__use
```
