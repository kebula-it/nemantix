# NXF005 — No Blank Line Before a Block Closer

**Category:** Blank lines  

## Rule

No blank line may appear immediately before a block closer (`__action`, `__frame`,
`__body`, `__repeat`, etc.).

## Rationale

A trailing blank line before a closer adds visual noise without conveying structure.
The blank-line separation between sections ([NXF004](NXF004.md)) already provides
sufficient visual grouping.

## Examples

**Compliant:**

```nxs
action greet >> greets the user <<:
  body:
    >> say hello <<
  __body
__action
```

**Non-compliant — blank line before `__body`:**

```nxs
action greet >> greets the user <<:
  body:
    >> say hello <<

  __body
__action
```

**Non-compliant — blank line before `__action`:**

```nxs
action greet >> greets the user <<:
  body:
    >> say hello <<
  __body

__action
```

## Autofix

Removes the blank line immediately before the block closer.
