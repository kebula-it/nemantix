# NXF002 — Line Length

**Category:** Line length  

## Rule

No line may exceed **120 characters**.

When a line exceeds this limit, the formatter switches to the multi-line form
of the construct (see [NXF201](NXF201.md), [NXF401](NXF401.md), [NXF402](NXF402.md)).

## Rationale

120 characters accommodate deep nesting with 2-space indentation ([NXF001](NXF001.md))
while remaining readable on standard widescreen displays without horizontal scrolling.

## Examples

**Compliant — prompt fits inline:**

```nxs
do llm using [[query]] >> summarise the content in a single concise sentence << producing [[result]]
```

**Compliant — prompt moved to block form because the inline version exceeds 120 characters:**

```nxs
do llm using [[query]] producing [[result]]
  >>> summarise the content of the document focusing on the key findings and omitting
      any introductory or concluding remarks <<<
```

## Notes

- The limit applies to the rendered line including leading indentation spaces.
- Blank lines and comment lines (`# ...`) are exempt.
